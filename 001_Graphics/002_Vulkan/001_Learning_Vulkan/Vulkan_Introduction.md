# Vulkan 简介
Vulkan 是由 Khronos Group 开发的跨平台、高性能、低开销的图形与计算 API（应用程序接口），作为 OpenGL 和 OpenCL 的现代继任者，于 2016 年正式发布。它被广泛用于游戏、虚拟现实、高性能渲染和并行计算等领域。

## 什么是 Vulkan？
底层图形 API：直接与 GPU 通信，提供对硬件的精细控制。
跨平台支持：支持 Windows、Linux、Android、macOS（通过 MoltenVK）等系统。
多语言支持：主要使用 C/C++，也有 Rust、Python 等语言绑定。
统一图形与计算：既能做 3D 渲染，也能执行通用 GPU 计算（GPGPU）。

核心设计理念
* 特性
** 显式控制:    开发者需手动管理内存、同步、资源生命周期，减少驱动开销
  低 CPU 开销: 减少驱动层的“黑箱”操作，提升多核 CPU 利用率
  多线程友好: 支持从多个线程并行录制命令，充分发挥现代 CPU 性能
  可预测性: 行为更一致，性能更稳定，适合高性能应用（如游戏引擎）
